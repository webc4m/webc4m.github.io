Install SWFOpenerSetup.exe and install_flash_player_ax.exe,
then open ea.comm.swf with UnH Solutions SWF Opener to run
the game! Optionally, you can run the beta version SWF in
the subfolder.


If you need to downgrade/reinstall the Flash Plugin,
just remove the key:

HKEY_LOCAL_MACHINE\Software\Macromedia (on a 32-bit OS)
or
HKEY_LOCAL_MACHINE\Software\Wow6432Node\Macromedia (on a 64-bit OS)
from your registry

and the installer will be able to run again.

Also tested and working under Wine!
